## Topics Covered

### 1. Cloudflare Moltworker (Serverless Middleware)
Cloudflare released Moltworker in January 2026, enabling OpenClaw to run serverless on their global edge network across 300+ data centers. Key features include Durable Objects for persistent state, R2 for storage, and the AI Gateway for model routing. Pricing starts at just $5/month with generous free tiers.

- https://blog.cloudflare.com/moltworker-self-hosted-ai-agent/
- https://www.infoq.com/news/2026/02/cloudflare-moltworker/
- https://singhajit.com/moltworker-self-hosted-ai-agent/

### 2. Security: Malicious Skills Crisis
The ClawHub ecosystem faced a major security issue with 341+ malicious skills delivering Atomic macOS Stealer (AMOS). Updated scans revealed over 800 malicious skills (~20% of the registry). Additional threats included infostealers targeting OpenClaw configurations and a malicious VS Code extension. The Dutch DPA issued warnings about deploying experimental agents on sensitive systems.

- https://www.malwarebytes.com/blog/news/2026/02/openclaw-what-is-it-and-can-you-use-it-safely
- https://www.acronis.com/en/tru/posts/openclaw-agentic-ai-in-the-wild-architecture-adoption-and-emerging-security-risks/

### 3. Peter Steinberger Joins OpenAI
On February 14, 2026, OpenClaw creator Peter Steinberger joined OpenAI, with the company setting up a foundation to support the OpenClaw project. Sam Altman described him as "a genius with a lot of amazing ideas about the future of very smart agents." Steinberger previously founded PSPDFKit in 2011.

- https://www.forbes.com/sites/ronschmelzer/2026/02/16/openai-hires-openclaw-creator-peter-steinberger-and-sets-up-foundation/

### 6. Anthropic Claude SDK Cost Reality

### 7. Prompt Injection & Consent Issues
TechWire Asia covered prompt injection going mainstream. The MoltMatch incident highlighted consent concerns when agents take actions without explicit authorization.

## Tips from This Episode
1. Start with Moltworker for cheap, global deployment ($5/mo)
2. Be VERY careful with third-party skills - audit before installing
3. Run security hardening updates immediately
4. Consider the consent implications of agent autonomy
5. The ROI on business automation use cases is compelling
