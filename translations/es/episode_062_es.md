[NOVA]: Anima de Fullive-AI es un Agent OS de código abierto para inteligencia de hardware, dirigido a dispositivos IoT, robótica y agentes de borde. [ALLOY]: Los agentes de codificación sufren cuando cada nueva sesión comienza con amnesia. El contexto solo de transcripción es frágil. [NOVA]: Un archivo .ctx.md es el camino intermedio ligero. [ALLOY]: El compromiso es que un archivo de contexto no es memoria inteligente por sí mismo. No sabe automáticamente qué importa. Puede quedarse obsoleto. Puede crecer demasiado.
