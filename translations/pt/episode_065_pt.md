[NOVA]: Um cartão de agente é um manifesto JSON que descreve o que um agente pode fazer. Quando um agente deseja delegar trabalho a outro, ele descobre o cartão do agente alvo e usa o cartão para descobrir como se comunicar.

[ALLOY]: A máquina de estados de tarefas é a outra metade do protocolo. A máquina de estados define o que cada estado significa, quais transições são válidas e quais dados os agentes trocam em cada transição. A máquina de estados é o que torna um fluxo de trabalho multiagente robusto contra falha parcial.
