[NOVA]: A biblioteca Transformers da Hugging Face lançou a versão 5.14 em 16 de julho com duas correções direcionadas para integração com Inkling. Transformers é um kit de ferramentas amplamente usado para carregar e executar modelos de pesos abertos, então correções estreitas de compatibilidade podem afetar muitos sistemas de inferência downstream mesmo quando o lançamento não adiciona nenhum recurso visível.

[ALLOY]: A primeira reparação cobre geração assistida quando um cache de encoder-decoder está presente. A geração assistida usa um modelo draft menor para propor tokens, então pede ao modelo maior para verificá-los. Aceitar vários tokens propostos pode acelerar a saída em comparação com gerar cada token independentemente. Um cache de encoder-decoder armazena as informações de atenção intermediárias usadas por modelos que processam entrada e saída através de estágios separados. O Inkling poderia falhar nesse caminho; a versão 5.14 o repara.

[NOVA]: A segunda correção se aplica durante o pré-preenchimento, quando um modelo processa o prompt antes de gerar uma resposta. Ela cobre o Inkling usando um cache estático, atenção scaled dot-product, entrada sem padding e viés de posição. Em linguagem comum, essa combinação reserva memória antecipadamente, executa um cálculo de atenção eficiente, evita adicionar tokens de preenchimento e ainda informa ao modelo onde os tokens pertencem. A interação anteriormente quebrava; o patch a restaura.

[ALLOY]: Estas são reparações de compatibilidade em vez de uma nova interface de programação. Elas importam para pessoas que fazem auto-hospedagem do Inkling porque duas configurações legítimas de inferência não travam mais. Elas também ilustram como integrações de modelo podem falhar nas interseções entre cache, kernels de atenção, formatos de entrada e manipulação específica de posição do modelo. Um checkpoint pode carregar corretamente e ainda assim quebrar apenas quando recursos de aceleração são combinados. O Transformers 5.14 fecha duas dessas interseções e dá aos serviços de inferência um caminho mais confiável para decodificação assistida e memória pré-alocada.
